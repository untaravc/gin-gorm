
# Golang Webserver Boiler Plate
Web app build with
- Golang
- Gin
- Gorm
- MySql connection

created from various tutorial on youtube

## Run Development
- Install Air for run development server

```
  go install github.com/air-verse/air@latest
  air server --port 8080
```

## Feature
- routing
- config db mysql
- middleware
- controller (CRUD)
- model
- log
